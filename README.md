# Web Customer Project 

### Summary 

Customer Relationship Manager.

The project is a Maven archetype for Spring MVC web application.

------------------
### Project characteristics

* Spring MVC
* JPA (Hibernate/MySQL)
* AOP (logging support)
 
 -----------------
### Prerequisites

* JDK 8 (tested with Java 11)
* Maven 3 (tested with 3.6.0)
